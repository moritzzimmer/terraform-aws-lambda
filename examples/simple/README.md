# Example without event

Creates an AWS Lambda function without an event trigger.

## usage

```
terraform init
terraform plan
terraform apply
```

Note that this example may create resources which cost money. Run `terraform destroy` to destroy those resources.

## Requirements

| Name | Version |
|------|---------|
| terraform | >= 0.12 |
| aws | >= 3.19 |

## Providers

No provider.

## Inputs

No input.

## Outputs

No output.
